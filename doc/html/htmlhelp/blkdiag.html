opSpot/blkdiag
 BLKDIAG   Block-diagonal concatenation of operator input arguments.
 
    B = blkdiag(OP1,OP2,...) produces the block-diagonal operator
 
             [ OP1                ]
         B = [      OP2           ]
             [           ...      ]
             [                OPN ]
See also
opSpot/diag, opBlockDiag.
